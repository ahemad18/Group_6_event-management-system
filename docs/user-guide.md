## Getting Started

### Opening the Application
1. Navigate to the project folder
2. Open `src/index.html` in your web browser
3. The application will load automatically

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, or Edge)
- JavaScript enabled
- Internet connection (for future API features)

## Features Overview

### 1. Event Browsing
- View all upcoming events in a card layout
- See event details including date, location, and capacity
- Check available spots in real-time

### 2. Search Functionality
- Search events by name, location, or description
- Real-time search results as you type
- Clear and intuitive interface

### 3. Event Registration
- Fill out a simple registration form
- Select from available events
- Validation ensures data accuracy
- Instant confirmation upon successful registration

## How to Use

### Browsing Events
1. Scroll to the "Upcoming Events" section
2. Browse through available events
3. Click on any event card to see more details

### Searching for Events
1. Use the search bar at the top of the page
2. Type keywords related to event name, location, or description
3. Results filter automatically as you type
4. Clear the search to see all events again

### Registering for an Event
1. Scroll to the "Register for an Event" section
2. Fill in your personal information:
   - Full Name (required)
   - Email Address (required)
   - Phone Number (required)
3. Select the event you want to attend
4. Add any additional comments (optional)
5. Click "Register Now" button
6. Wait for confirmation message

### Registration Tips
- Ensure your email address is correct for confirmation
- Phone number should include area code
- Register early as spots are limited
- Check available spots before registering

## Troubleshooting

### Form Won't Submit
- Check that all required fields are filled
- Ensure email format is correct (example@domain.com)
- Verify phone number has at least 10 digits
- Make sure an event is selected

### Events Not Showing
- Refresh the page
- Clear browser cache
- Ensure JavaScript is enabled
- Check browser console for errors

### Search Not Working
- Try different keywords
- Check spelling
- Use simpler search terms
- Refresh the page if needed
